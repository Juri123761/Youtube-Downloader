# YouTube Downloader

A simple desktop application for downloading YouTube videos as MP4.

## Download

Download the installer file: `YouTubeDownloader_Setup.exe`

## Installation

1. Download `YouTubeDownloader_Setup.exe`
2. Run the installer
3. Choose whether to create a desktop shortcut

## System Requirements

- Windows 10 or Windows 11
